Html Emmets
Emmets Description
! or html:5 Generates a basic HTML5 template.
> The > character is used to create child elements.
+ The + character is used to create sibling elements.
* To create Multiple elements
{} we simply wrap the content in curly braces
^ It add the element to the parent
[ ] It is used to add attributes to HTML elements.
. It is used to add class names to HTML elements.
( ) The parentheses () are used for grouping elements.
$ The $ symbol is used as a placeholder for numbering elements
