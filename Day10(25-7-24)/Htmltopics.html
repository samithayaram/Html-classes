HTML
  • Hyper text markup language
  • It is used to create a webpages.
  • It acts as a structure to website
Markup It is set of rules which is used to structure the content,images and layout.Markup refers to a system of annotating a document in a way that is syntactically distinguishable from the text.
Hypertext It text refferal to another text.Hypertext refers to a system of organizing and displaying text in a non-linear fashion, allowing for easy navigation and interaction through embedded links.

Types of Tags
  • Container tags or pair tags
  • Empty tags or void tags or self closing tags
Anatomy of element
  • Element is nothing but it having opening tag, content and closing tag
  • Opening Tag: This marks the beginning of an element and includes the element's name inside angle brackets.
  • Content: This is the text or other HTML elements nested inside the element.
  • Closing Tag: This marks the end of an element and is similar to the opening tag but includes a forward slash before the element's name.
Series of element A series of HTML elements refers to multiple HTML tags used in sequence to create a structured and formatted web document. These elements can be nested within each other and can include a variety of tags to add headings, paragraphs, links, images, lists, tables, and other content.
Nested elements Nested elements in HTML refer to placing one or more elements inside another element. This allows for creating complex structures and hierarchies, which help in defining the layout and organization of web content.
